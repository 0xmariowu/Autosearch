## Notes

- The schema has no dedicated field for free-form strategy notes such as "queries come from `queries.json`" or "apply engagement filter during harvest", so those details are encoded in `phases[*].query_source` and the strategy-specific section overrides.
- The JSON files are complete genomes instead of minimal patches because `load_genome()` only backfills some sections from dataclass defaults; notably, `modes` must be present for validation.
