Why AutoSearch

Self-evolving

Learns which queries and channels work for each topic. Session 3 is measurably better than session 1.

Gap-first search

Claude knows 60-70% of most topics already. AutoSearch only searches for what's missing.

Chinese internet

12 native Chinese channels: Zhihu, Bilibili, CSDN, WeChat, Weibo, and more. No other tool covers this.

Cited reports

Two-stage citation lock. Every claim is traced to a source. Markdown, HTML, or slides.

Fixed evaluator

The judge scores results on 7 dimensions. Evolution cannot modify the evaluator. No gaming.

Zero config

Install and run. All channels work without API keys. Add Exa or Tavily optionally for semantic search.
