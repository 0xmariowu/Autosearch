## When to use

Use when the task needs research papers or preprints and arXiv is likely to be a primary source.

## Quality signals

- Strong title match and relevant abstract text
- Recent publication timestamps
- Recognizable authors in the target area

## Known limits

- arXiv is not the same as Papers With Code and has no code metadata here
- Search is based on simple term expansion over a few query words
- Metadata quality varies by paper entry
