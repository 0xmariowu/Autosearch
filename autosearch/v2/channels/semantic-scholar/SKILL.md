## When to use

Use when the query is research-oriented and paper discovery is more important than general web coverage.

## Quality signals

- Strong title and abstract match
- Citation counts that suggest impact
- Author lists and publication year aligned with the topic

## Known limits

- API rate limiting can cause retries or empty results
- Some papers have incomplete metadata
- Citation mode depends on resolving the initial paper correctly
