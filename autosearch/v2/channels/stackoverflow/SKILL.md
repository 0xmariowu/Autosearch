## When to use

Use when the query looks like an error, exception, code symptom, library usage question, or practical implementation problem.

## Quality signals

- Strong title match against the error or API name
- Relevant tags for the language or framework
- High score and answered status

## Known limits

- Snippets are synthetic summaries rather than full question bodies
- Results are limited to Stack Overflow, not the full Stack Exchange network
- Popular older answers can outrank newer edge cases
