# Platform

G2 — business software review platform with verified user reviews. Provides ratings, pros/cons, and feature comparisons. Essential for product evaluation tasks.

# When To Choose It

Choose this when:

- evaluating software products from a user perspective
- need verified user reviews (not marketing copy)
- comparing competing products in a category
- want satisfaction scores and feature ratings

# How To Search

- `site:g2.com {product or category keywords}`

Example queries:
- `site:g2.com AI agent platform review`
- `site:g2.com AI research tool comparison`
- `site:g2.com "AI coding assistant" review 2026`

# Standard Output Schema

- `source`: `"g2"`

# Date Metadata

G2 reviews have dates. Extract from snippet.

# Quality Bar

This skill is working when it discovers real user sentiment and product comparisons that vendor marketing and tech blogs don't provide honestly.
