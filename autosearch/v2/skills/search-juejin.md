# Platform

Juejin (掘金) — Chinese technology blog and community platform. Strong in frontend, backend, AI/ML, and DevOps content. Known for high-quality original technical articles.

# When To Choose It

Choose this when:

- need Chinese-language deep dives on specific technologies
- searching for Chinese developer opinions on tools and frameworks
- the topic is trending in Chinese tech community
- need Chinese translations or explanations of English-language concepts

# How To Search

- `site:juejin.cn {Chinese keywords}`

Example queries:
- `site:juejin.cn 自进化 AI agent 2026`
- `site:juejin.cn LLM 工具调用 best practice`
- `site:juejin.cn MCP 协议 实战`

# Standard Output Schema

- `source`: `"juejin"`

# Date Metadata

Juejin articles show creation dates. Extract from snippet text.

# Quality Bar

This skill is working when it discovers original Chinese technical content with practical insights not found on English platforms.
