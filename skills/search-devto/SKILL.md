# Platform

Dev.to — community platform for developer blog posts. Practical tutorials, experience reports, and opinion pieces. More practitioner-focused than academic sources.

# When To Choose It

Choose this when:

- need practical tutorials and how-to articles
- looking for developer opinions and experience reports
- searching for beginner-friendly explanations of complex topics
- want community discussion around a technology choice

# How To Search

- `site:dev.to {keywords}`

Example queries:
- `site:dev.to self-evolving AI agent building`
- `site:dev.to LLM agent framework comparison 2026`
- `site:dev.to Claude Code skills tutorial`

# Standard Output Schema

- `source`: `"devto"`

# Date Metadata

Dev.to articles show publication dates. Extract from snippet.

# Quality Bar

This skill is working when it discovers practitioner perspectives and tutorials that academic searches and GitHub READMEs don't provide.
